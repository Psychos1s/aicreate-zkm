## 实例与模型对象的语义对应关系(🔴 仅上下文中包含「需智」关键词的时候采用)

在解析知识文档时，需要通过语义判断明确表头字段与需智模型对象之间的对应关系。以下映射表列举了常见字段及其可能的语义映射，帮助避免歧义性理解：

| 文档字段（常见变体） | 对应模型对象 | 说明 |
|---------------------|-------------|------|
| **模糊意图 / 需求名称 / 需求类型 / 用户诉求 / 业务需求 / 问题类型** | 【模糊的意图】对象 | 需求名称 → 需求描述 |
| **诊断指标 / 检测指标 / 评估指标 / 衡量指标 / 关键指标 / KPI / 核心指标 / 业务指标 / 绩效指标** | 【需求价值指标】- 诊断参数 + 【特征标签】对象 | 用于判断需求可行性时需要采集的诊断参数，可能同时关联需求价值指标和特征标签 |
| **可行性判定规则 / 判定条件 / 阈值条件 / 成立条件 / 触发条件 / 准入条件 / 筛选条件** | 【需求价值指标】- 诊断规则 | 判定指标成立的阈值或条件 |
| **价值目标 / 需求领域 / 价值维度 / 价值方向 / 收益目标 / 业务目标 / 预期效果 / 达成目标** | 具象化为【需求价值指标】对象 | 表达不同维度的需求价值指标，将价值目标内容拆解为具体指标 |
| **需要的能力 / 能力要求 / 解决方案 / 应对策略 / 实施路径 / 技术方案 / 服务能力 / 核心能力** | 【需要的能力】对象 | 满足需求所需的能力 |
| **引导问题 / 询问问题 / 诊断问题 / 探查问题 / 提问话术** | **忽略（不生成到模型文档）** | 辅助信息，用于引导追问环节，不直接对应模型对象 |
| **诊断话术示例 / 话术模板 / 沟通示例 / 参考话术** | **忽略（不生成到模型文档）** | 辅助信息，用于引导追问环节，不直接对应模型对象 |
| **目标客群 / 客户类型 / 适用对象 / 目标客户 / 服务对象 / 业务主体 / 用户群体** | 【目标客群】对象 | 有某种共同分类特征的客户群体 |
| **痛点问题 / 客户痛点 / 核心问题 / 关键挑战 / 难点分析 / 需求背景** | 【模糊的意图】- 需求描述 | 体现客户的痛点问题 |
| **业务场景 / 应用场景 / 使用场景 / 典型场景 / 典型案例 / 行业场景** | 【目标客群】- 与模糊意图的关联 或 **忽略** | 用于理解需求背景；典型案例类内容不直接映射到模型对象 |
| **成功案例 / 标杆案例 / 参考案例 / 实施案例 / 项目案例** | **忽略** | 案例信息不直接对应模型对象，可作为背景参考 |
| **关键要素 / 核心要素 / 重要因素 / 影响因素 / 决定因素** | 【需求价值指标】- 诊断参数 或 【特征标签】对象 | 影响需求诊断的关键因素 |
| **约束条件 / 限制因素 / 制约因素 / 边界条件 / 前提条件** | 【特征标签】- 对需求价值指标的约束 | 对需求实现的限制条件 |
| **风险因素 / 风险点 / 潜在风险 / 不确定因素** | 【特征标签】- 对需求价值指标的约束 | 可能影响需求实现的风险 |
| **行业特点 / 行业特征 / 业务特征 / 领域特性 / 专业属性** | 【特征标签】对象 或 **忽略** | 行业通用特征可映射为特征标签 |
| **参考标准 / 规范要求 / 行业标准 / 技术规范 / 合规要求** | 【需求价值指标】- 诊断规则 | 判定指标成立的标准依据 |
| **成本构成 / 费用明细 / 投入成本 / 投资预算 / 成本预估** | 【需求价值指标】- 诊断参数 或 【需要的能力】- 能力要素 | 与成本相关的信息 |
| **实施步骤 / 操作流程 / 实施流程 / 作业流程 / 工作步骤** | 【需要的能力】- 能力方案（价值流） | 能力的执行方法与步骤 |
| **注意事项 / 警示说明 / 使用须知 / 操作规范 / 安全要求** | 【特征标签】- 对需求价值指标的约束 或 **忽略** | 安全或规范类要求可作为约束条件 |
| **关键词 / 标签 / 分类标签 / 索引标签 / 标记词** | 【特征标签】对象 | 用于分类或标记的标签信息 |
| **关联关系 / 关联规则 / 依赖关系 / 上下游关系** | 模型对象之间的关系 | 描述对象之间的关联关系 |
| **计量单位 / 单位说明 / 数量单位** | 【需求价值指标】- 诊断参数 | 明确参数的计量单位 |
| **数据来源 / 数据渠道 / 信息来源 / 采集方式** | 【需求价值指标】- 诊断参数 | 诊断参数的数据源 |
| **时效性 / 有效期 / 适用期限 / 更新周期** | 【需求价值指标】- 诊断参数 | 参数的时序范围或有效期 |
